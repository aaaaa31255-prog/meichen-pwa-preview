# Privacy Preview

This repository is a public sanitized preview for Meichen Beauty AI PWA.

## What this preview contains

- Static demo pages only
- Mock/demo descriptions
- Public-facing product structure

## What this preview does not contain

- Real customer names, phone numbers, membership balances, or salon records
- API keys, tokens, passwords, secrets, or `.env` files
- Private project logs, automation logs, or project_hub records
- Original brand color-chart PDFs, training videos, or unlicensed source materials
- Full private business rules or production formula datasets

## Data behavior

This public preview is intended for static GitHub Pages display. It should not collect, store, or transmit real customer information.

Production privacy rules are handled separately in the private development repository and platform-specific release packages.
